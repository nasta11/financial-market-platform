# Financial Market Analysis Platform

## Описание проекта

Financial Market Analysis Platform — прототип информационно-аналитической платформы для анализа финансовых рынков.

Система позволяет получать биржевые данные из открытых источников, сохранять их в базе данных PostgreSQL, выполнять аналитическую обработку, строить графики и формировать прогнозы на основе методов машинного обучения.

## Используемые технологии

* Python 3.12
* FastAPI
* PostgreSQL
* SQLAlchemy
* Pandas
* Plotly
* yfinance
* Scikit-learn

## Основные функции

### Загрузка данных

Получение исторических котировок финансовых инструментов через Yahoo Finance.

Пример:

POST /load/AAPL

### Просмотр данных

Получение сохранённых котировок из базы данных.

Пример:

GET /prices/AAPL

### Аналитика

Расчёт статистических показателей:

* средняя цена;
* максимальная цена;
* минимальная цена;
* объём торгов.

Пример:

GET /analytics/AAPL

### Визуализация

Построение интерактивных графиков изменения стоимости финансового актива.

Пример:

GET /chart/AAPL

### Dashboard

Комплексная аналитическая панель с показателями и графиком.

Пример:

GET /dashboard/AAPL

### Прогнозирование

Прогноз цены на следующий торговый день с использованием линейной регрессии.

Пример:

GET /predict/AAPL
